Installation

Clavier+ est disponible en deux versions :

  • Avec programme d’installation : http://utilfr42.free.fr/dn/ClavierSetup.exe
  • Sans programme d’installation : http://utilfr42.free.fr/dn/Clavier.zip

Pour installer Clavier+ sans programme d’installation :

  1. Téléchargez le fichier Zip et décompressez-le.
  2. Lancez Clavier.exe.
  3. Si vous souhaitez que les raccourcis soient effectifs en permanence, relancez Clavier.exe pour afficher la fenêtre de configuration, puis cochez la case Lancer Clavier+ au démarrage de Windows.

Pour désinstaller Clavier+ sans programme d’installation :

  1. Dans la configuration de Clavier+, décochez si nécessaire la case Lancer Clavier+ au démarrage de Windows.
  2. Quittez Clavier+ en cliquant sur le bouton à ce nom.
  3. Supprimez les fichiers de Clavier+.

Ce que permet de faire Clavier+

Clavier+ permet d’associer une action à n’importe quel raccourci clavier. Ces actions peuvent être :

  • lancer un programme : Word, calculatrice, navigateur Internet, etc.
  • afficher un site Internet
  • ouvrir un dossier : Mes documents, C:, lecteur de CD-Rom, etc.
  • écrire des caractères spéciaux : majuscules accentuées, symboles ® et ©, etc.
  • écrire du texte : adresse e-mail, formule de politesse, etc. dans n’importe quel programme (Word, navigateur Internet, bloc-notes)
  • simuler des frappes de touches, notamment pour exécuter une commande située dans le menu d’un programme

Les raccourcis clavier peuvent utiliser (presque) toutes les touches du clavier. La touche Windows (à gauche et à droite de la touche Espace) peut être employée en complément de Ctrl, Alt et Maj. Comme les logiciels utilisent rarement cette touche, de nombreux raccourcis clavier simples sont disponibles : Win + A, Win + B, Win + C, etc.

Clavier+ peut également modifier le comportement des touches spéciales comme ² ou F11. On peut ainsi associer à une action fréquence un raccourci clavier à une seule touche.

Si nécessaire, l’action d’un raccourci clavier peut dépendre du programme actif. Cela permet d’ajouter des raccourcis clavier à un programme en particulier.

De même, l’action d’un raccourci clavier peut dépendre de l’état des touches bascules, c’est-à-dire les touches comportant des petites lumières sur le clavier : verrouillage majuscule, pavé numérique, arrêt défilement. Cela permet, par exemple, d’écrire les majuscules accentuées au lieu des chiffres quand le verrouillage majuscule est activé.

Le site de Clavier+ propose quelques exemples de raccourcis.

Créer un raccourci clavier

Lancez Clavier+. Une icône en forme de clavier apparaît à côté de l’horloge : elle indique que Clavier+ est lancé, donc que les raccourcis clavier sont actifs. Cliquez sur l’icône pour afficher la fenêtre de configuration. Vous pouvez redimensionner la fenêtre si vous la trouvez trop petite. La zone supérieure contient la liste des raccourcis créés. Plusieurs sont créés initialement, à titre d’exemple.

Pour créer un nouveau raccourci, cliquez sur le bouton Ajouter (en forme de +), et choisissez l’action du raccourci clavier :

  • Programme : exécuter un programme du menu Démarrer
  • Favori Internet : afficher un site Internet référencé dans les favoris d’Internet Explorer (si vous utilisez un autre navigateur, par exemple Firefox, choisissez Afficher un site Internet plus bas dans le menu)
  • Caractère spécial : écrire un caractère spécial
  • Dossier : ouvrir un dossier dans l’explorateur
  • Écrire du texte : simuler la frappe d’un texte, avec éventuellement l’exécution de raccourcis clavier intermédiaires (par exemple la touche Tab pour remplir d’un seul coup tous les champs d’un formulaire)
  • Lancer un programme : lancer un programme absent du menu Démarrer, ou exécuter une ligne de commande quelconque
  • Afficher un site Internet : afficher un site Internet absent des favoris d’Internet Explorer

Ensuite, une boîte de dialogue apparaît pour saisir le raccourci clavier. Effectuez votre raccourci clavier dans la zone prévue à cet effet : son nom devrait s’écrire dans la zone. Cliquez enfin sur OK.

Si nécessaire, modifiez l’action du raccourci dans la moitié inférieure de la boîte de dialogue. Si le raccourci doit écrire du texte, par exemple une formule de politesse, vérifiez que l’option Écrire est cochée puis entrez le texte voulu dans la zone située juste en bas. À chaque fois que le raccourci sera effectué, le texte spécifié sera écrit. Vous pouvez utiliser le bouton en forme de flèche pour insérer des caractères spéciaux ou simuler un raccourci clavier.

Si le raccourci doit lancer un programme ou afficher un site Internet, cochez l’option correspondante. Entrez le chemin d’accès du programme ou l’adresse du site Web dans la zone. Vous pouvez cliquer sur le petit bouton situé à droite pour chercher le programme à lancer. Vous pouvez vérifier que le programme ou le site Internet est correct avec le bouton Tester. Si nécessaire, cliquez sur le bouton Paramètres avancés pour régler le répertoire de démarrage du programme ou son mode d’affichage (plein écran, fenêtré, réduit).

Important : vous devez fermer la fenêtre de Clavier+ avant de tester vos raccourcis clavier. Tant que la fenêtre est visible, les raccourcis clavier sont désactivés.

Conditions sur les raccourcis

Vous pouvez limiter la reconnaissance du raccourci à certaines conditions. Si toutes les conditions sont vérifiées, l’action associée au raccourci est exécutée. Si une ou plusieurs conditions ne sont pas remplies, Clavier+ ignore le raccourci, et l’action originale du raccourci est effectuée (celle de Windows ou du programme actif). Cela permet de donner des significations différentes au même raccourci clavier.

Attention ! Si vous créez plusieurs entrées pour le même raccourci, veillez à leur affecter des conditions mutuellement exclusives. Dans le cas contraire, un message d’erreur s’affichera avant la validation.

Programme actif

Vous pouvez restreindre un raccourci à certains programmes. Cela permet de donner des significations différentes au même raccourci clavier, selon le programme courant.

Pour définir la liste des programmes pour lesquels le raccourci clavier est activé, sélectionnez le raccourci dans la liste. Ensuite, cliquez sur le viseur noir sur fond blanc situé en bas à droite, et laissez le bouton de la souris enfoncé. Déplacez le viseur sur la fenêtre du programme voulu (par exemple, une fenêtre de Word ou de l’explorateur). Le nom du programme s’inscrit dans la zone située à gauche du viseur. Répétez l’opération pour chaque programme à ajouter.

Quand liste de programmes est vide, le raccourci est actif quel que soit le programme courant. Quand la zone n’est pas vide, le raccourci clavier n’est actif que pour les programmes spécifiés. Il est possible d’ajouter plusieurs programmes à la liste, séparés par des points-virgules.

État des touches bascule

Vous pouvez modifier l’action d’un raccourci selon l’état des touches bascule : verrouillage majuscule, pavé numérique, arrêt défilement. Vous pouvez par exemple créer un raccourci qui ne sera actif que si le mode verrouillage majuscule est activé.

Pour cela, double-cliquez sur le raccourci souhaité dans la liste, ou bien sélectionnez-le et cliquez sur le bouton en forme de crayon. Dans la zone Conditions d’activation, définissez la condition portant sur chaque touche bascule. Vous pouvez imposer que la touche soit activée, désactivée, ou bien ne pas imposer de condition.

Format du texte écrit par les raccourcis

Dans la zone de texte Écrire, vous pouvez ordonner l’exécution d’autres raccourcis clavier. Pour cela il suffit d’écrire le raccourci clavier entre crochets, à l’endroit où il doit être effectué. Par exemple :
[Ctrl + O]

Pour connaître le nom des touches spéciales, vous devez recopier le nom du raccourci tel qu’indiqué lors de l’ajout d’un raccourci. Par exemple, pour connaître le nom de la touche étoile du pavé numérique, cliquez sur le bouton en forme de + puis appuyez sur la touche étoile. Son nom s’affiche alors: * (PAVE NUM.). Il faudra donc écrire [* (PAVE NUM.)] pour simuler l’appui de cette touche. Veillez bien à conserver les symboles spéciaux du nom de la touche: points, parenthèses, accents, etc.

Si vous avez besoin d’insérer le caractère [ sans marquer le début d’un raccourci, vous devez le faire précéder d’un antislash. Par exemple :
Ceci est un crochet ouvrant: \[

De même, pour entrer le caractère antislash, vous devez le mettre en double : Voici un seul antislash: \\

Clavier+ envoie les raccourcis à la fenêtre active au début. Si vous demandez l’exécution d’un raccourci qui change la fenêtre active (par exemple [Ctrl + O], vous devez ordonner à Clavier+ de se focaliser sur la nouvelle fenêtre active. Pour cela, insérez le raccourci vide : []. Clavier+ fera une pause de 100 millisecondes et recherchera la nouvelle fenêtre active. Exemple :
[Ctrl + O][][][]MonFichier.txt

Pour exécuter un programme au milieu d’une saisie de texte, écrivez le chemin d’accès du programme entre doubles crochets :
[Ctrl+C][[notepad.exe]][][Ctrl+V]
Ce raccourci copie le texte actuellement sélectionné dans le presse-papiers, lance le bloc-notes, puis y colle le texte copié. Le [] est très important, il indique à Clavier+ de coller dans la nouvelle fenêtre active, c’est-à-dire celle du bloc-notes.

Imprimer la liste des raccourcis clavier

Dans la fenêtre de Clavier+, cliquez sur le bouton Copier la liste. Cela place la liste des raccourcis clavier dans le presse-papiers : vous pouvez ensuite la coller dans le logiciel de votre choix pour l’imprimer. Si vous utilisez Word, vous pouvez la mettre sous forme de tableaux : lancez Word, collez la liste avec Ctrl+V, sélectionnez-là avec Ctrl+A, puis cliquez sur le bouton Insérer un tableau.

Fichier de configuration

Le fichier Clavier.ini contient tous les paramètres. Le programme ne met rien dans la base de registre, à l’exception de l’option Lancer Clavier+ au démarrage de Windows.

Si vous voulez ajouter d’un seul coup de nombreux raccourcis, voici le format du fichier de configuration :

Paramètres globaux
Paramètres du premier raccourci
-       (ligne presque vide, ne comportant qu’un tiret)
Paramètres du second raccourci
-
...

Paramètres globaux

Langue=Français ou English ou Deutsch
Taille=largeur,hauteur,agrandie,masquer icône
Colonnes=largeur 1,largeur 2,largeur 3
Taille
largeur,hauteur : taille de la fenêtre de configuration, en pixels.
agrandie : 1 si la fenêtre est agrandie à tout l’écran, 0 sinon.
masquer icône : 1 pour masquer l’icône de Clavier+ à côté de l’horloge, 0 pour la rendre visible.
Colonnes
Indique la largeur des trois premières colonnes dans la fenêtre de configuration. Chaque largeur est un pourcentage relatif à la largeur totale de la liste. Elles sont séparées par des virgules. La dernière colonne occupe l’espace restant.

Paramètres des raccourcis

Raccourci=séquence de touches
Code=code de touche
Description=texte de description
Texte=texte à taper quand le raccourci est effectué
>ligne supplémentaire
>autre ligne supplémentaire
...
Commande=programme à lancer quand le raccourci est effectué
Répertoire=répertoire de démarrage
Fenêtre=mode de fenêtre du programme à lancer
Programmes=liste des programmes
VerrMaj=condition
VerrNum=condition
ArrêtDéfil=condition

Notez la nécessité du caractère > au début des lignes supplémentaires pour le champ Texte=.

Les options Texte d’une part, Commande, Répertoire et Fenêtre d’autre part sont mutuellement exclusives : un raccourci ne peut pas à la fois entrer du texte et lancer un programme.

Raccourci
Représentation textuelle des touches du raccourci clavier, au même format que dans la fenêtre de paramétrage.
Code
Représentation numérique des touches du raccourci clavier, utilisé en interne par Clavier+. Ce code n’est utilisé que si le champ Raccourci est erroné.
Répertoire (mode commande uniquement)
Répertoire de démarrage de la commande. Si aucun répertoire n’est spécifié, Clavier+ lance le programme dans le répertoire de son exécutable.
Fenêtre (mode commande uniquement)
Taille initiale de la fenêtre du programme lancé, parmi : Normale, Réduite, Agrandie.
Programmes
Condition d’activation du raccourci : liste des seuls programmes pour lesquels le raccourci est activé. Si la liste est vide, le raccourci est actif dans tous les programmes. Les programmes sont séparés par des points-virgules, et n’ont pas de chemin d’accès : nom seul, avec extension.
VerrMaj, VerrNum, ArrêtDéfil
Si présents, définissent une condition d’activation du raccourci. Valeurs possibles : Aucune (valeur par défaut), Oui et Non.

Limitations

Certaines touches spéciales ne peuvent être utilisées :

Impr écran
Pause

Les touches spéciales rafraîchir le navigateur, augmenter le volume, etc. sont partiellement supportées. On peut créer des raccourcis les utilisant, et même modifier leur comportement d’origine. Cependant, ceci peut surprendre, l’action de ces touches s’exécute lorsqu’on les enfonce dans la boîte de dialogue Ajouter. De plus, comme Windows ne propose pas de méthode générique pour récupérer le nom de ces touches spéciales, elles s’affichent sous la forme d’un numéro (par exemple : #183).

Les raccourcis standards de Windows ne peuvent être modifiés. En voici une liste non exhaustive :

Win + Elance l’explorateur
Win + Faffiche la fenêtre Rechercher des fichiers
Win + Ctrl + Faffiche la fenêtre Rechercher des ordinateurs
Win + Lchangement rapide d’utilisateur (Windows XP)

FAQ

Retrouvez la dernière version de la FAQ sur la page de Clavier+.

Clavier+ ne fonctionne pas avec OpenOffice !

OpenOffice est un programme Java qui gère le clavier de manière non-standard. Pour tout de même créer des raccourcis fonctionnant sous OpenOffice, vous pouvez :

  • Mettre le nom des touches entre crochets, par exemple [T][O][T][O] pour toto, ou [Maj+T][Maj+O][Maj+T][Maj+O] pour TOTO. Notez qu’il faut écrire Maj+Lettre pour les lettres majuscules.
  • Utiliser les fonctionnalités d’OpenOffice pour créer des raccourcis clavier. Le tutoriel de Framasoft vous y aidera.
Comment écrire de majuscules accentuées dans OpenOffice ?

La technique des crochets, [É], ne fonctionne pas, car les crochets décrivent une touche physique (ici, la touche é/2/~ du clavier français), or cette touche ne permet pas d’écrire un É. Si l’on utilise [Maj+é] ou [Maj+É], cela écrira un 2.

La seule solution simple consiste à créer un raccourci clavier dans OpenOffice lui-même. Le tutoriel de Framasoft vous y aidera.

Clavier+ fonctionne-t-il avec les jeux ?

La plupart du temps, non. Les jeux ont un accès bas niveau sur le clavier, pour diverses raisons (rapidité, anti-triche, …).

Comment faire écrire un retour à la ligne par un raccourci ?

Deux solutions :

  • insérer un retour à la ligne dans la zone de texte, en appuyant sur la touche Entrée ;
  • écrire le code [ENTREE] (sans accent !).
Comment simuler la touche Machin ?

Cliquez sur le bouton en forme de flèche, commande Simuler un raccourci clavier.

Comment simuler un clic droit ?

Parfois, simuler la touche Menu Contextuel fonctionne. Avec la commande Simuler un raccourci clavier du bouton en forme de flèche, on constate que le code de cette touche est #93. Il suffit alors d’écrire [#93] pour simuler un clic-droit.

Comment changer le volume ?

Il existe trois touches relatives au son :

  • muet : code [#173]
  • baisser le volume : code [#174]
  • monter le volume : code [#175]

Vous pouvez également utiliser le programme VolumeManager pour régler plus finement le volume.

La case Lancer Clavier+ au démarrage de Windows ne fonctionne pas !

Créez un raccourci vers Clavier.exe dans Menu démarrer/Programmes/Démarrage. Pour cela, cliquez avec le bouton droit sur Clavier.exe, laissez le bouton enfoncé et faites glisser la souris successivement sur le menu démarrer, puis Programmes, puis Démarrage, et relâchez le bouton une fois dans le menu. Choisissez Créer un raccourci ici.

Comment masquer l’icône de Clavier+ à côté de l’horloge ?

Lancez Clavier+ puis quittez-le. Double-cliquez sur le fichier Clavier.ini situé dans le même répertoire que Clavier+. À la deuxième ligne, qui commence par Taille= remplacez le dernier zéro en un. Par exemple : Taille=923,501,0,0 doit devenir Taille=923,501,0,1.

Comme l’icône de Clavier+ ne sera plus visible, vous devrez relancer Clavier+ pour afficher la fenêtre de configuration.

Comment lancer deux programmes à la fois ?

La solution est de mettre le chemin d’accès des deux programmes entre double-crochets, dans la zone Écrire. Par exemple, pour lancer le bloc-notes et la calculatrice :

[[notepad.exe]][[calc.exe]]
Comment faire des choses plus compliquées : écrire la date du jour, activer une application… ?

Vous pouvez créer un script VBScript et demander à Clavier+ de l’exécuter. La procédure générique est la suivante : lancez le bloc-notes, écrivez le texte de votre script, et enregistrez-le avec l’extension .VBS, puis associez-lui un raccourci-clavier avec Clavier+.

Quelques exemples de scripts :

  • Écrire la date du jour dans le programme actif :
    set shell = WScript.CreateObject("WScript.Shell")
shell.SendKeys Date()
  • Activer une application et lui envoyer des touches, par exemple calculer 2+7 avec la calculatrice (qui doit être déjà lancée) :
    set shell = WScript.CreateObject("WScript.Shell")
shell.AppActivate "Calculatrice"
WScript.Sleep(100)  ' Attendre 100 millisecondes
shell.SendKeys "2{+}"
WScript.Sleep(100)
shell.SendKeys "7{=}"

    Consultez la documentation Microsoft pour savoir comment envoyer toutes les combinaisons de touches, y compris des raccourcis clavier compliqués.

Comment réellement simuler un clic gauche ou un clic droit ?

Là, il faut sortir l’artillerie lourde : appeler les fonctions bas niveau de Windows. On pourrait certes programmer un programme spécifique, mais le plus simple est d’utiliser un VBScript.

Il faut commencer par enregistrer une DLL qui permettra d’appeler les fonctions de Windows à partir d’un VBScript : téléchargez la DLL, copiez-la dans C:\Windows\System32 puis faites : Démarrer/Exécuter, regsvr32 dynwrap.dll puis validez. Un message devrait s’afficher indiquant que la DLL a été correctement enregistrée.

Écrivez ensuite le VBScript suivant, par exemple sous le nom click.vbs :

Set UserWrap = CreateObject("DynamicWrapper")
UserWrap.Register "user32.dll", "mouse_event", "i=lllll", "f=s", "r=l"
UserWrap.mouse_event 2, 0, 0, 0, 0
UserWrap.mouse_event 4, 0, 0, 0, 0

Ce script simule l’enfoncement puis le relâchement du clic gauche de la souris. Écrivez les nombres 8 et 16 au lieu de 2 et 4 dans les deux lignes commençant par UserWrap.mouse_event pour simuler un clic droit. Vous pouvez appeler le script avec Clavier+, ou bien insérer ce code dans un autre VBScript. Voyez la documentation officielle de Microsoft pour la référence complète de la fonction mouse_event, ou bien ce site qui a l’avantage de donner la valeur numérique des constantes, en hexadécimal.
